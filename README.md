VoltDB Example: Geo-Fencing 
============================

To compile and start the database in the background

    cd db
    ./run.sh startdb

To run the client:

    cd client
    ./run.sh client

To stop the database:

    cd db
    ./run.sh stopdb
     -or-
    voltadmin shutdown
    
To clean up this directory

    ./run.sh clean
